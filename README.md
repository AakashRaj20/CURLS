# CURLS
## Accepting for Hacktoberfest 2023
**Curls is text editor combination of <b>VSCode and Notepad</b> where you can code with unlimited people as in a room with suggestion like in a INTERVIEW**

## Tech Used
- **HTML**
- **Javascript**
- **CSS**
- **REACT**
- **WEB-SOCKETS**
